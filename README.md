# TabTrail

TabTrail 是一个 Chromium 浏览器插件，用来记录最近操作的浏览器标签，并把当前打开的标签按最近活跃、最近关闭、手动分类和网站域名快速整理。

## 核心功能

- 新标签页仪表盘：打开默认新标签页时展示 TabTrail 仪表盘，直接搜索、切换分类和整理标签。
- 分类整理：支持全部标签、手动分类、网站域名分类和最近关闭分类。
- 最近活跃：按时间查看最近打开、激活过的标签，并快速切回原标签。
- 最近关闭：查看最近关闭的标签并重新打开；这个分类是只读内容，不能拖入、编辑或关闭分类。
- 新建分类：选中标签后输入分类名，可以把标签归入新的手动分类。
- 拖拽归类：把当前打开的标签拖拽到可写的手动分类标题上完成归类。
- 批量关闭：可以关闭已选标签，也可以在确认后关闭当前普通分类中的全部打开标签。
- 侧边栏：保留独立侧边栏入口，适合在浏览网页时持续整理标签。
- 首次安装引导：第一次安装后打开 TabTrail 新标签页工作台，说明整理范围、侧边栏入口和最近关闭恢复；引导可跳过，完成或跳过后只展示一次。

## 安装

1. 在项目目录执行 `npm run build`。
2. 打开 Chrome 的 `chrome://extensions/` 或 Edge 的 `edge://extensions/`。
3. 开启“开发者模式”或“开发人员模式”。
4. 点击“加载已解压的扩展程序”或“加载解压缩的扩展”。
5. 选择构建生成的 `dist/` 目录。
6. 打开一个默认新标签页，确认展示 TabTrail 新标签页仪表盘。
7. 首次安装时确认新手引导可见，点击“开始使用 TabTrail”或“跳过”后不再自动展示。

## 使用方式

1. 打开新标签页进入新标签页仪表盘。
2. 使用顶部搜索框按标题、网址或域名过滤标签。
3. 点击分类标题切换当前分类；一行会展示多个分类标题，窄屏会自动换行。
4. 勾选标签后输入分类名，点击“归类已选”完成分类整理。
5. 将标签拖拽到手动分类标题上，可以快速移动到该分类。
6. 在普通分类中点击“关闭分类”，确认后关闭该分类中的当前打开标签。
7. 在最近关闭分类中点击记录，可以重新打开对应 URL。

## 权限说明

- `tabs`：读取当前打开标签的标题、URL、窗口和顺序，用于最近活跃、分类整理、切换标签和批量关闭。
- `storage`：在浏览器本地保存最近记录、手动分类、排序偏好和插件内固定状态。
- `sidePanel`：提供 TabTrail 侧边栏入口。
- `sessions`：在本地缓存不足时读取浏览器最近关闭会话，用于恢复最近关闭标签。

## 隐私边界

- 不上传最近活动、最近关闭或分类数据。
- 不读取网页正文内容。
- 不接入外部服务器。
- 不追踪完整浏览历史。
- 不修改默认搜索引擎、主页或地址栏搜索行为。
- 首次安装引导只说明 TabTrail 工作台能力，不上传标签数据，也不会修改默认搜索引擎。

## 浏览器限制

TabTrail 可以把默认新标签页替换为新标签页仪表盘，但不能直接隐藏浏览器原生顶部标签栏。这是 Chromium 扩展 API 的边界。

如果想恢复浏览器默认新标签页，可以在扩展管理页面禁用扩展；禁用扩展后浏览器会回到原来的默认新标签页体验。
