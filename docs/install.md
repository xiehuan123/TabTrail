# TabTrail 本地安装说明

## 构建

```bash
npm run build
```

构建产物会生成在 `dist/` 目录。该目录可以作为未打包扩展加载到 Chromium 浏览器。

## Chrome 加载方式

1. 打开 `chrome://extensions/`。
2. 开启右上角“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择本项目的 `dist/` 目录。
5. 点击浏览器工具栏中的 TabTrail 图标测试 popup。
6. 在扩展详情中打开侧边栏入口，测试 side panel。

## Edge 加载方式

1. 打开 `edge://extensions/`。
2. 开启“开发人员模式”。
3. 点击“加载解压缩的扩展”。
4. 选择本项目的 `dist/` 目录。

## 手动验证清单

- 打开新标签后，最近活动记录增加。
- 切换标签后，popup 最近活跃区域更新。
- 关闭标签后，popup 和 side panel 的“最近关闭”出现该标签。
- 从 popup 点击最近活跃标签，可以切回原标签。
- 从 popup 点击最近关闭标签，可以重新打开 URL。
- side panel 默认展示当前窗口，切到“全部窗口”后展示所有窗口标签。
- 搜索能按标题、URL、域名过滤。
- side panel 内上移/下移只改变预览顺序。
- 点击“应用到标签栏”后，真实标签栏顺序改变。
- 关闭 1-2 个标签不弹确认；关闭 3 个及以上弹确认。
- TabTrail 固定只影响插件展示，不改变浏览器原生 pinned 状态。
