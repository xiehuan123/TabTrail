# TabTrail 隐私边界

TabTrail MVP 只在浏览器扩展本地环境内工作，不包含服务器、账号系统或外部上传逻辑。

## 本机保存

以下数据只写入 `chrome.storage.local`：

- 最近打开标签记录
- 最近激活标签记录
- 最近关闭标签记录

最近活动记录最多保留 100 条。

## 浏览器同步

以下轻量数据写入 `chrome.storage.sync`：

- 手动分类元数据
- TabTrail 内固定状态
- 预览排序偏好
- 视图偏好

最近打开、激活、关闭活动记录不会写入同步存储。

## 不做的事情

- 不读取网页正文内容。
- 不追踪完整浏览历史。
- 不上传最近活动记录。
- 不接入外部服务器。
- 不改变浏览器原生 pinned 状态。
